# Gemelo Digital de Planeación

Aplicación desarrollada en Python con Dash para la planeación, desagregación y simulación de producción.

## 🚀 Funcionalidades

- Planeación agregada (optimización con PuLP)
- Desagregación por producto
- Simulación de eventos discretos (SimPy)
- KPIs de producción
- Análisis de escenarios

## 🛠️ Tecnologías

- Python
- Dash
- Pandas
- NumPy
- SimPy
- PuLP
- Plotly
